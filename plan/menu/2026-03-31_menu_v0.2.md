# Menu Color Scheme — Simplified Plan

## Ý tưởng

Thêm **duy nhất 1 prop** `colorScheme` trên `<Menu>`. Nó cascade xuống toàn bộ children (MenuItem, MenuSubTrigger, MenuSubContent, MenuLabel, MenuDivider...) thông qua Context hiện có.

**Nguyên tắc:**
- Value là **raw CSS color string** (`'#1565c0'`, `'rgba(255,255,255,0.12)'`, `'white'`...) — không phụ thuộc theme
- Field nào **không điền** → lấy default hiện tại của menu (các giá trị hardcode trong `styled.tsx`)
- **Không có** sub-level override — tất cả lấy từ container

---

## Interface

```typescript
export interface MenuColorScheme {
  /** Menu container background */
  background?: string;
  /** Default text color */
  color?: string;

  /** Hover state */
  hoverBg?: string;
  hoverColor?: string;

  /** Active/selected item */
  activeBg?: string;
  activeColor?: string;

  /** Soft-selected (parent trigger of active child) */
  softSelectedBg?: string;

  /** Danger item */
  dangerColor?: string;
  dangerHoverBg?: string;

  /** Disabled text */
  disabledColor?: string;

  /** Secondary text (labels, shortcuts, arrows) */
  secondaryColor?: string;

  /** Divider line */
  dividerColor?: string;

  /** Focus ring color */
  focusRingColor?: string;

  /** Popover sub-menu background (nếu mode='popover') */
  popoverBg?: string;
  /** Popover border color */
  popoverBorderColor?: string;
}
```

## DX — Cách sử dụng

```tsx
// Sidebar xanh dương, chữ trắng — chỉ cần fill cái muốn đổi
<Menu
  colorScheme={{
    background: '#1565c0',
    color: '#fff',
    hoverBg: 'rgba(255,255,255,0.12)',
    activeBg: 'rgba(255,255,255,0.24)',
    secondaryColor: 'rgba(255,255,255,0.7)',
    dividerColor: 'rgba(255,255,255,0.2)',
    focusRingColor: '#fff',
  }}
>
  <MenuItem icon={<HomeIcon />}>Home</MenuItem>
  <MenuItem icon={<UsersIcon />} selected>Users</MenuItem>
  <MenuSub>
    <MenuSubTrigger icon={<SettingsIcon />}>Settings</MenuSubTrigger>
    <MenuSubContent>
      <MenuItem>General</MenuItem>
      <MenuItem>Security</MenuItem>
    </MenuSubContent>
  </MenuSub>
</Menu>

// Không truyền colorScheme → hoạt động y hệt hiện tại, zero breaking change
<Menu>
  <MenuItem>Normal menu</MenuItem>
</Menu>
```

### Styled
#### [MODIFY] [styled.tsx](file:///d:/workspace/react-lib-workspace/libs/menu/src/lib/styled.tsx)

Mỗi styled component nhận thêm prop `ownerColorScheme?: MenuColorScheme`. Logic:

```typescript
// Ví dụ cho MenuItemStyled
const textColor = ownerDanger
  ? (ownerColorScheme?.dangerColor ?? palette?.error?.main ?? '#d32f2f')
  : ownerDisabled
    ? (ownerColorScheme?.disabledColor ?? palette?.text?.disabled ?? 'rgba(0,0,0,0.38)')
    : (ownerColorScheme?.color ?? palette?.text?.primary ?? 'rgba(0,0,0,0.87)');

// Hover
'&:hover': {
  backgroundColor: ownerDanger
    ? (ownerColorScheme?.dangerHoverBg ?? palette?.error?.light ?? '...')
    : (ownerColorScheme?.hoverBg ?? palette?.action?.hover ?? '...'),
  ...(ownerColorScheme?.hoverColor && { color: ownerColorScheme.hoverColor }),
}
```

Priority chain: **`colorScheme` → `palette` (theme) → hardcoded fallback**

Các styled cần update:

| Styled component | Tokens dùng |
|---|---|
| `MenuContainerStyled` | `background`, `color` |
| `MenuItemStyled` | `color`, `hoverBg`, `hoverColor`, `activeBg`, `activeColor`, `softSelectedBg`, `dangerColor`, `dangerHoverBg`, `disabledColor`, `focusRingColor` |
| `MenuItemShortcutStyled` | `secondaryColor` |
| `MenuLabelStyled` | `secondaryColor` |
| `SubArrowStyled` | `secondaryColor` |
| `MenuDividerStyled` | `dividerColor` |
| `PopoverSubContentStyled` | `popoverBg`, `popoverBorderColor` |
